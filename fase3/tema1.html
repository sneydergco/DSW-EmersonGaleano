Interfaz de Usuario

Proposito

User Interface se centra en la parte visual. Es decir, si UX se encarga de que un producto sea útil para los usuarios, UI lo hace atractivo y visual. Hace parte del UX.

Es importante aclarar que Diseño de Interfaces no equivale a Diseño Gráfico: el diseño de la interfaz puede incluir o no diseño gráfico. Por ejemplo, cuando se realiza un wireframe está diseñando una interfaz pero no está aplicando diseño gráfico, y cuando se aplican reglas de estilo a una interfaz se está aplicando diseño gráfico pero no se está diseñando una interfaz.
